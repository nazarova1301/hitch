# Тестирование веб-сервиса Hitch :briefcase:

Я протестировала веб-сервис Hitch, предназначенный для поиска сотрудников и размещения анкет кандидатов, и подготовила различные виды документации по тестированию:

* [Тест-кейсы 1](https://docs.google.com/spreadsheets/d/1-VE4TnkEMW-Y8WrXs5URyh7mnpcub918JQZBHoiE_4o/edit?usp=drive_link) – исследовательское тестирование форм регистрации и авторизации с помощью техник тест-дизайна: граничных значений и классов эквивалентности.
* [Тест-кейсы 2](https://docs.google.com/spreadsheets/d/1T_ejEGEwsdfJOdbCqkfy8WRLp9-qGo_gVmmcnHk4F7E/edit?usp=drive_link) – кроссбраузерное тестирование личного кабинета кандидата и компании.
* [Тест-кейсы 3](https://docs.google.com/spreadsheets/d/1XLQ4bKg7HL1S6zhCtqpOl-hepqGdpmOfd4i4c7YVWNE/edit?usp=drive_link) – функциональное тестирование формы «Новая заявка».
* [Тест-кейсы 4](https://docs.google.com/spreadsheets/d/14JgmneuXMAvfZjbtV_mj8E5An9M8gXcA-ocHYIRXHNE/edit?usp=drive_link) – тестирование новой функциональности, повторное и регрессионное тестирование.
* [Матрица трассировки](https://docs.google.com/spreadsheets/d/1bVM5o8rxlFZrkfETj_NrDj1PObMTo4SJlPg8El5h5Fk/edit?usp=drive_link) – покрытие тестами всех ключевых требований.
* [Баг-репорты](https://ru.yougile.com/board/gcvg2dir1ioy) созданы с помощью TMS YouGile.
* [Отчёт о тестировании](https://docs.google.com/document/d/1eeVX1zEs5wxe8fW3mISLXeQgzXxETZxIkjV_qP503Rk/edit?usp=drive_link) составлен после завершения тестирования.
